My Projects

Word Clock

An Arduino based project that shows you the time using words.

Word Clock

This project of mine was to build a clock that told you the time using words. I wrote the code for this project in Arduino and ran the program through the Arduino itself. Along with telling the time, I included functionality to change the colour displayed, change the time and tell the temperature.

Music Box

An RFID controlled music playing box, that allows you to switch between playlists by simply placing a different card on the Music Box.

Music Box

This project of mine was to build a a device that allowed you to play different music playlists by simply placing an RFID card on top of a wooden box. I wrote the code for this project in Python and ran the script on the startup of a Raspberry Pi. I programmed the device to automatically play the playlist corresponding to the card used, I also made a simple graphical interface that displayed the current song name, artist name and the album art of the song being played. Added functionality included in this device was buttons that allowed the user to pause/play, skip, rewind and change the volume of songs being played.

Height Estimation App

A web app that allows you to calculate the height of objects using your phone camera.

Height Estimation App

This app was built as a project in my Engineering Mobile Apps unit at university. The task was to create and app that used the in-built hardware on a mobile phone to calculate the distance to and the height of an object. Using trigonometry the app calculates the height of and the height to the base of any object. This is done by using your camera to lock in where the base and apex of the object lies, using readings of the accelerometer of your phone, the angle of your phone is calculated and then the height of the object.

Automatica

1st Place Hardhack 2018.

Automatica

This was my team's submission to the 2018 Monash 24-hour Hardhack Competition. We decided to build an automated locker system during the overnight hacking competition. This locker was designed to automate the transfer of goods, when buying and selling items between university students. As university students we knew all too well the difficulty of organising the buying and selling of textbooks between university students. The main issue we tried to address in our design was the issue of finding a common time and place to meetup and handover the textbook from seller to buyer. This automatic locker system allowed for the seller to drop off the item in their own time (tracked seller based on Student ID card) and then have the buyer pick up the item when they feel like (scan ID card and locker associated with the sale automatically opens). This project of ours was awarded 1st place at the 2018 Monash Hardhack Competition.

Puck-Stacking Robot

As part of the ECE3091 Engineering Design project, our team was tasked with creating a robot that could avoid obstacles in an arena, find pucks in the arena and bring them back to a designated area for stacking.
